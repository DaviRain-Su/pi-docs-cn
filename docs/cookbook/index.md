# Cookbook

> 代码与场景化实践汇总

本目录汇总可直接落地的开发场景与实战片段，适合直接复制到扩展里用。

## 目录

- [第11章：Cookbook](/cookbook/11-cookbook)

## 推荐阅读顺序

1. 先看 `11-cookbook`，熟悉工具、命令、事件与 UI 样例。
2. 将示例按你的项目场景抽象为“扩展模板”。
3. 结合 `reference/extensions.md` 的 API 说明，把 `cookbook` 示例逐步替换为自己的能力。

## 常用入口

- 会话内代码片段：`/cookbook/11-cookbook`
- 扩展开发 API：[`/reference/extensions`](/reference/extensions)
- 常见技能组合：[`/reference/json-mode`](/reference/json-mode)
