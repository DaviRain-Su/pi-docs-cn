## 一分钟入门

1. 确认 shell 与 Node.js 版本（至少支持 npm 安装）。
2. 按 `terminal-setup` 完成基础依赖安装。
3. 用 `tmux` 建立稳定会话。
4. 在需要的设备上跑一次 `npm install && npm run check`（视仓库要求）。
