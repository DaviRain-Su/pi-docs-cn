# dAgent X Layer 项目

> 去中心化 AI Agent 协作网络 - 基于 ERC-8004 标准的多链 Agent 市场

---

## 项目简介

dAgent 是**首个基于 ERC-8004 标准的多链 AI Agent 市场**，让用户可以雇佣专业 AI 团队完成从 DeFi 交易到智能合约开发的全方位任务。

### 核心特性

- 🎯 **8 大类 Agent**：金融交易、开发技术、内容创作、数据分析、安全审计、运营自动化、社交互动、通用工具
- ⛓️ **多链支持**：X Layer 主场 + Solana + Sui + Ethereum
- 📜 **ERC-8004 标准**：首个完整实现，统一 Agent 身份与声誉
- 💰 **USDC 结算**：即时支付，透明定价，无订阅锁定
